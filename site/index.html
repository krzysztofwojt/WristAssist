FAQ

Q Does Nadgar support real-time voice mode?

A No. I built a proof of concept, but on Apple Watch it is not practical right now. Real-time voice requires low-level networking, which is not allowed by default on watchOS. See Apple's TN3135: Low-level networking on watchOS.

Q Are there any ads?

A No.

Q Is the app free to use?

A The app is free to use. You still need an OpenAI Platform account and your own API key. OpenAI charges you for API usage.

Q Can I use my ChatGPT or Codex subscription with Nadgar?

A Not at this time. ChatGPT and Codex subscriptions do not provide a general OpenAI API key for apps like Nadgar. I may revisit this if OpenAI offers a clean way to support it.

Q Do you store my API key on your servers?

A No. Nadgar connects only to OpenAI servers. Your API key is stored locally on your iPhone and Apple Watch in Keychain.

Q How is it different from Siri?

A Older Apple Watches do not have Siri on-device. Nadgar lets you use more capable AI models with web search and, later, connect your own MCP servers from your wrist.

Privacy Policy for Nadgar

Nadgar is an Apple Watch and iPhone app published by Krzysztof Wójt.

Effective date: June 28, 2026

Information Nadgar Uses

Nadgar is a bring-your-own OpenAI API key app. The app may use:

  • OpenAI API key: entered on iPhone, stored locally in Apple Keychain, and synced to the paired Apple Watch so the Watch can make OpenAI API requests.
  • Microphone audio: recorded only when you use push-to-talk on Apple Watch.
  • Transcription and chat content: your recorded audio is sent to OpenAI for transcription, and the resulting text and chat context are sent to OpenAI to generate an assistant response.
  • Text-to-speech content: if speech playback is enabled, assistant text may be sent to OpenAI to generate audio playback.
  • App settings: model, voice, transcription, speech, and related preferences are stored locally on your devices.

Nadgar does not require you to create a Nadgar account.

How Information Is Stored

Your OpenAI API key is stored locally in Apple Keychain on your iPhone and, after sync, on your paired Apple Watch.

Temporary audio files are created on Apple Watch during push-to-talk recording and are deleted after the request is completed, cancelled, or cleaned up by the app.

Nadgar does not operate its own server for storing your API key, audio, transcripts, or chat messages.

Information Sent to OpenAI

When you use Nadgar, the app sends requests directly from your Apple Watch or iPhone to OpenAI services using the OpenAI API key you provide. These requests may include your API key, microphone audio, transcribed text, chat messages, assistant instructions, selected model settings, and text used for speech playback.

OpenAI processes this information under OpenAI's API terms and privacy policies. You should review OpenAI's policies to understand how OpenAI handles API data.

Information Nadgar Does Not Collect

Nadgar does not collect or use data for advertising or tracking. Nadgar does not access your location, contacts, calendar, photos, health data, camera, or advertising identifier.

Nadgar does not include third-party analytics or advertising SDKs.

Diagnostics

Nadgar may write local diagnostic logs such as request status, model names, durations, byte counts, and error messages to help troubleshoot app behavior. These logs are intended not to include your OpenAI API key, raw audio, or full chat content.

Apple may separately collect crash reports or diagnostics according to your Apple device settings and Apple's privacy policies.

Children's Privacy

Nadgar is not intended for children under 13. The app does not knowingly collect personal information from children.

Changes to This Policy
